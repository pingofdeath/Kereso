El símbolo tipográfico " - " puede referirse a:

  • El símbolo ASCII "-", código Unicode U+002D o &#45; que tiene el doble valor de guión y signo menos;
  • ‐ el guión ortográfico, un símbolo de puntuación, código Unicode U+2010 o &#8208;
  • − el signo menos, el sígno para representar la sustracción aritmética, código Unicode U+2212 o &#8722.
